Trilha Back-End

Fundamentos

Neste módulo, você ira aprender os conceitos básicos necessários para começar a desenvolver no lado do servidor.

Introdução a Servidores e Bancos de Dados:

  • Noções básicas sobre como servidores funcionam.
  • Introdução a bancos de dados relacionais e não relacionais.

Linguagens de Programação Server-Side:

  • Exploração de linguagens como Node.js (JavaScript), Python, Ruby, etc.
  • Configuração básica do ambiente de desenvolvimento.

Bancos de Dados Relacionais:

  • Introdução a SQL (Structured Query Language).
  • Modelagem básica de dados e relacionamentos.

APIs (Interfaces de Programação de Aplicações):

  • Compreensão dos conceitos de API.
  • Construção de APIs simples usando frameworks como Express (Node.js) ou Flask (Python).

Intermediário

O segundo módulo aprofunda os conhecimentos, introduzindo conceitos mais avançados e práticas comuns em desenvolvimento back-end.

Frameworks e Bibliotecas:

  • Exploração de frameworks back-end populares, como Express (Node.js), Flask (Python), Django (Python), Ruby on Rails (Ruby), etc.
  • Utilização de bibliotecas para facilitar o desenvolvimento.

Autenticação e Autorização:

  • Implementação de sistemas de autenticação..
  • Controle de acesso e autorização de usuários.

Bancos de Dados NoSQL:

  • Introdução a bancos de dados NoSQL (por exemplo, MongoDB).
  • Modelagem de dados NoSQL.

Middleware e Roteamento:

  • Compreensão de middleware em frameworks back-end.
  • Implementação de roteamento avançado.

Avançado

O terceiro módulo abrange desafios mais complexos e a criação de sistemas back-end robustos e eficientes.

Arquiteturas Back-End Avançadas:

  • Padrões de arquitetura como RESTful, GraphQL, e Microservices.
  • Estratégias para escalabilidade.

Segurança Back-End:

  • Práticas de segurança em desenvolvimento back-end.
  • Proteção contra ameaças comuns, como injeção de SQL.

Testes e Qualidade de Código:

  • Implementação de testes unitários e de integração.
  • Ferramentas de teste e boas práticas de codificação.

Integração com Front-End:

  • Construção de APIs para comunicação eficiente com o front-end.
  • Utilização de autenticação JWT (JSON Web Tokens) para segurança.
