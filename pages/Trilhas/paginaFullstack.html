Trilha Full-Stack

Fundamentos

Neste módulo inicial, você ira adquirir as habilidades básicas necessárias para entender tanto o front-end quanto o back-end.

HTML, CSS e JavaScript:

  • Revisão dos conceitos fundamentais do desenvolvimento front-end.
  • Construção de páginas web interativas.

Linguagem de Programação (Exemplo: JavaScript, Python, c#):

  • Introdução à linguagem de programação para desenvolvimento tanto no front-end quanto no back-end.
  • Manipulação do DOM e execução de scripts no servidor (Node.js, por exemplo).

Noções Básicas de Bancos de Dados:

  • Introdução a bancos de dados relacionais e NoSQL.
  • Execução de operações CRUD (Create, Read, Update, Delete).

Integração Front-End e Back-End:

  • Construção de APIs simples para serem consumidas no front-end.
  • Compreensão básica da comunicação entre front-end e back-end.

Intermediário

O segundo módulo aprofunda os conhecimentos, permitindo que os alunos construam aplicações mais complexas e integradas.

Frameworks Front-End e Back-End:

  • Exploração de frameworks back-end populares, como Express (Node.js), Flask (Python), Django (Python), Ruby on Rails (Ruby), etc.
  • Utilização de frameworks back-end populares, como Express (Node.js), Django (Python), etc.

Bancos de Dados Avançados:

  • Modelagem avançada de dados.
  • Otimização de consultas e transações.

Autenticação e Autorização Completa:

  • Implementação de sistemas de autenticação seguros.
  • Controle de acesso e autorização em diferentes partes da aplicação.

Desenvolvimento Responsivo e Mobile:

  • Construção de aplicações responsivas para diversos dispositivos.
  • Introdução ao desenvolvimento mobile.

Avançado

O terceiro módulo prepara você para assumir projetos completos, compreendendo a arquitetura e a segurança de ponta a ponta.

Arquitetura Full-Stack:

  • Implementação de arquiteturas avançadas como Microservices ou Serverless.
  • Estratégias para integração eficiente entre front-end e back-end.

Segurança Completa da Aplicação:

  • Práticas de segurança em todas as camadas da aplicação.
  • Proteção contra ameaças comuns em ambientes web.

Otimização de Desempenho:

  • Estratégias para otimização de desempenho tanto no front-end quanto no back-end.
  • Utilização de ferramentas para análise de desempenho.

Projeto Full-Stack:

  • Desenvolvimento de um projeto completo, desde a concepção até a implementação.
  • Implementação de boas práticas de desenvolvimento colaborativo.
