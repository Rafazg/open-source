Trilha Front-End

Fundamentos

Neste módulo, você ira aprender os conceitos básicos essenciais para construir uma base sólida no desenvolvimento front-end. Aqui estão alguns tópicos que seram abordados:

HTML (HyperText Markup Language):

  • Estrutura básica de uma página HTML.
  • Tags e elementos HTML.
  • Formulários e elementos de entrada.

CSS (Cascading Style Sheets):

  • Estilização de elementos HTML.
  • Seletores CSS.
  • Layouts básicos e posicionamento
  • Responsividade e design para dispositivos móveis.

JavaScript (JS):

  • Introdução à programação JavaScript.
  • Manipulação do DOM (Document Object Model).
  • Eventos e interatividade.
  • Noções básicas de AJAX para carregamento assíncrono.

Controle de Versão:

  • Uso básico de sistemas de controle de versão, como Git.
  • Colaboração em projetos usando plataformas como GitHub.

Intermediário

No segundo módulo, você irá aprofundar seus conhecimentos e habilidades, explorando tópicos mais avançados e práticos.

Frameworks Front-End:

  • Introdução a frameworks populares, como React, Angular ou Vue.js.
  • Compreensão de componentes e estado.
  • Roteamento e manipulação de dados.

CSS Avançado:

  • Pré-processadores CSS (Sass ou Less).
  • Animações e transformações CSS.
  • Metodologias de organização de estilos, como BEM (Block, Element, Modifier).

JavaScript Avançado:

  • Programação assíncrona e Promises.
  • Manipulação avançada do DOM.
  • Introdução a ES6+ (Arrow functions, classes, destructuring, etc.).

Acessibilidade e Performance:

  • Desenvolvimento focado em acessibilidade.
  • Otimização de desempenho para carregamento rápido de páginas.

Avançado

O terceiro módulo visa preparar você para desafios complexos e projetos mais avançados.

Arquitetura de Aplicações Front-End:

  • Padrões de design (por exemplo, MVC, MVVM).
  • Estruturação de projetos escaláveis.

Testes e Qualidade de Código:

  • Testes unitários e de integração.
  • Ferramentas de teste como Jest.
  • Boas práticas de codificação e revisão de código.

Segurança Front-End:

  • Práticas de segurança em aplicações web.
  • Proteção contra ataques comuns, como Cross-Site Scripting (XSS) e Cross-Site Request Forgery (CSRF).

Integração com Back-End:

  • Compreensão de como interagir com APIs RESTful.
  • Autenticação e autorização.
  • Uso de bibliotecas como Axios para requisições HTTP.
